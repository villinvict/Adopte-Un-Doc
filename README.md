# Adopte Un Doc
ING 2 semester 1 project at EISTI

## Serveur
```sh
cd server
# Installer les dépendances
npm install

# Démarrer le serveur local
npm start
```

## Client
```sh
cd client
# Installer les dépendances
npm install

# Démarrer le client localement et ouvrir la version web
expo start --web
```
